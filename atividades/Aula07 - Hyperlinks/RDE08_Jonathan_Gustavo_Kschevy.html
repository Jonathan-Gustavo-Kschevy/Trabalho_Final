table:

A tag < table > serve para criar tabelas no documento HTML. Dentro dela, são inseridos os elementos que compõem a tabela.

del:

A tag < del > representa uma parte do texto excluída do documento.

ins:

A tag < ins > é utilizada quando um texto inserido na página foi modificado.

sub:

A tag < sub > é utilizada para escrever um texto com letras menores e alinhamento abaixo do restante do conteúdo.

sup:

A tag < sup > é utilizada para escrever um texto com letras menores e com o alinhamento acima do restante do conteúdo.

blockquote:

A tag < blockquote > é utilizada para indicar que o texto incluído é uma longa citação.

q:

A tag < q > é utilizada para indicar que o texto incluído é uma pequena citação.

abbr:

A tag &l abbr > representa uma abreviação , podendo fornecer uma descrição completa à ela.

address:

A tag &l address > pode ser utilizada para fornecer o contato de uma empresa, aoutor etc.

cite:

A tag &l cite > serve para referenciar uma trabalho artístico. Deve conter o título do trabalho ou uma URL de referência.

bdo:

A tag &l bdo > serve para substituir a direcionalidade atual do texto.

code:

A tag &l code > é usada para definir uma parte do código do computador.

pre:

A tag &l pre > pode ser utilizada para representar um texto pré-formatado.

kbd:

A tag &l kbd > é utilizada para orientar o usuário em instruções, geralmente, on-line, porém também podem ser impressas.
