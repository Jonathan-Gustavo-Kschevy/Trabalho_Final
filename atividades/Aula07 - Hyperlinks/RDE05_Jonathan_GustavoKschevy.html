Marcação de Hiperlinks:

Pode ser feita utilizando a tag < a > e a propriedade href, que referencia ao conteúdo desejado.

Sintaxe:

< a href="endereço" > Conteúdo Referenciado Clicável < /a >

Hiperlinks Externos ou Remotos:

Utiliza-se a URL no campo da propriedade href.

Sintaxe:

< a href="URL" > Conteúdo Referenciado < /a >

Hiperlinks Internos:

Apontam para um lugar diferente dentro da mesma página.

Sintaxe:

< a href="#referencia" > Conteúdo Referenciado < /a >

< a name="referencia" > < /a >

< a href="#referencia" > Conteúdo Referenciado < /a >

< tag id="referencia" > < /a >

Hiperlinks Locais:

Direcionam para outras páginas dentro do mesmo site.

Sintaxe:

< a href="arquivo.html" > Conteúdo Referenciado < /a >

Hiperlinks em Imagens:

Envolve-se a tag de imagem com a tag de hiperlink.

Sintaxe:

< a href="referencia" > < img src="imagem.jpg" > < /a >

Hiperlinks para Endereço Eletrônico:

Direcionam para um endereço eletrônico.

Sintaxe:

< a href="mailto:email@email.com" > email@email.com < /a >
